Essential HTML Tags

Below are 10 HTML tags used to structure web content.

1. <h1> to <h6>

Used for headings to create a hierarchy.

<h1>Main Title</h1>

2. <p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

3. <a>

Defines a hyperlink.

<a href="index.html">Home</a>

4. <img>

Embeds an image using the src attribute.

<img src="image.jpg" alt="Description">

5. <ul> & <li>

Creates a bulleted list.

<ul><li>Item</li></ul>

6. <ol>

Creates a numbered list.

<ol><li>Step 1</li></ol>

7. <div>

A container for grouping elements.

<div>Content</div>

8. <code>

Displays a single line of code.

<code>CSS property</code>

9. <pre>

Preserves spaces and line breaks for code blocks.

<pre>Formatted text</pre>

10. <footer>

Defines the bottom section of a page.

<footer>Copyright info</footer>
