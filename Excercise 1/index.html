HTML & CSS Resource

Here are 10 examples of HTML tags, and 8 CSS properties

10 Examples of HTML Tags

1. <h1> to <h6>

Used for headings. h1 is the most important, h6 the least.

<h1>Main Title</h1>

2. <p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

3. <a>

The anchor tag creates a hyperlink to another page.

<a href="https://google.com">Link text</a>

4. <ul>

Creates an unordered (bulleted) list.

<ul><li>Item</li></ul>

5. <ol>

Creates an ordered (numbered) list.

<ol><li>Step 1</li></ol>

6. <img>

Embeds an image into the document.

<img src="image.jpg" alt="Description">

7. <strong>

Indicates that the text has strong importance (renders as bold).

<strong>Important text</strong>

8. <em>

Indicates emphasis (renders as italic).

<em>Emphasized text</em>

9. <br>

Inserts a single line break.

Line one <br> Line two

10. <div>

A container used to group elements together for styling.

<div class="container">...</div>

8 Essential CSS Properties

1. color

Sets the color of the text.

p { color: red; }

2. background-color

Sets the background color of an element.

body { background-color: lightblue; }

3. font-size

Changes the size of the text.

h1 { font-size: 32px; }

4. margin

Creates space outside the border of an element.

div { margin: 20px; }

5. padding

Creates space inside the element, between content and border.

div { padding: 10px; }

6. border

Sets a border around an element.

img { border: 2px solid black; }

7. text-align

Sets the horizontal alignment of text.

h1 { text-align: center; }

8. width

Sets the width of an element.

section { width: 50%; }
